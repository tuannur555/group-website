WELCOME TO BASIC COMPUTER KNOWLEDGE LEARNING

Here's six modules in which users can learn of computer.

The module will tell you the definition of computer as well as how the computer works (Information Processing Cycle).

How To Set Up Computer

The module contains the step of setting up a computer that are purchase in the box that will helps in the use of computer.

The module consists of the basic computer command or shortcut keys to increase productivity and saves users time.

How To Connect To A Network
