INTRODUCTION TO COMPUTER



What is Computer?

Computer is a technology that are build to perform processes, calculations and operations according to the instruction provided by software and hardware program. It also used to store data for later uses in appropriate storage devices and we can retrieve them whenever it is necessary. Besides that, computer are designed to execute applications and provide solutions by combining hardware and software component.

Nowadays, computer are used for a variety of purposes such as browsing the web, editing videos, playing video games, creating document, creating applications and many more.






How Computer System works?

A computer system allows data to be process, store and produce information. The series of input, processing, output and storage is referred to the information processing cycle. In order to understand the cycle, situation are given to explain how the sytem works. Let's say that a clerk is registering new students using computer.


1

INPUT

The clerk enters the student's details such as name and address using input devices which is keyboard and digital scanner.

2

PROCESSING

The computer uses stored instruction to process the data into information.The component involve in this process are motherboard and processor.


3

OUTPUT

Make this yours. Add images, text and links, or connect data from your collection.

4

STORAGE

The data and information are stored to temporary memory and then on a permanent storage devices such as hard drives, memory card, optical disk and USB flash drive.






What is the Characteristic of Computer?




1

SPEED

The computer can process data very fast, at the rate of millions of instructions per second. Some calculations that would have taken hours and days to complete otherwise, can be completed in a few seconds using the computer.

2

ACCURACY

Computer provides a high degree of accuracy. For example, the computer can accurately give the result of division of any two numbers up to 10 decimal places.

3

DILIGENCE

When used for a longer period of time, the computer does not get tired or fatigued. It can perform long and complex calculations with the same speed and accuracy from the start till the end.

4

STORAGE CAPABILITY

Data and information can be stored and retrieved whenever required. A limited amount of data can be stored, temporarily, in the primary memory. Secondary storage devices like floppy disk and compact disk can store a large amount of data permanently.

5

VERSATILITY

Computer is versatile in nature. It can perform different types of tasks with the same ease. At one moment you can use the computer to prepare a letter document and in the next moment you may play music or print a document.
